## About

RenCricut is a small Pesach shop I started at 16. Every product is designed in Cricut Design Space and made by hand — vinyl labels, fabric bags, paper goods, and more. This site is where you can browse the collection and place a custom order before Pesach.

Everything is made to order, so your family name, your words, and your details go on each piece.

## Running Locally

```bash
git clone https://github.com/yehoshuk/RenCricut.git
cd RenCricut
npx serve .        # Node
# or
python -m http.server  # Python 3
```

Open http://localhost:3000
